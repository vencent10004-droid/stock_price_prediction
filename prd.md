# PRD — Product Requirements Document
# StockSense: AI 주가 예측 & 애널리스트 리포트 자동화

**버전:** 2.0  
**작성일:** 2026-06-26  
**상태:** 확정 기획안

---

## 1. 배경 및 문제 정의

### 현재 문제
개인 투자자가 매일 직면하는 3가지 핵심 어려움:

| 문제 | 현실 |
|------|------|
| **방향 판단** | 오늘 주가 보고 내일 올라갈지 내려갈지 판단이 어려움 |
| **가격대 불확실성** | 올라간다 해도 얼마까지? 내려간다면 어디서 지지될까? |
| **정보 과잉** | 뉴스·리포트가 너무 많아 핵심만 빠르게 파악하기 어려움 |

### 해결책
AI가 **오늘 데이터를 분석해 내일 방향과 가격대를 예측**하고,  
이를 **전문 애널리스트 수준의 보고서**로 자동 생성해 이메일로 발송

---

## 2. 대상 사용자

### 페르소나 A: 개인 투자자 "박준호"
- 국내 주식 3~5종목 보유
- 매일 아침 방향 판단이 어려움
- **원하는 것:** "내일 삼성전자 살까요 팔까요?" 한 줄 답변 + 근거

### 페르소나 B: 소규모 투자팀 리더 "이서연"
- 팀원들에게 매일 아침 시황 보고 필요
- 직접 리포트 쓰는 시간 절약하고 싶음
- **원하는 것:** 전문가처럼 보이는 일일 리포트를 자동으로 팀에 전송

## 3. 핵심 기능 요구사항

### F-01. 다종목 주가 데이터 수집
- 시가총액 상위 10개 기업 등 **관심 종목 목록 설정** (`config.yaml`)
- OHLCV + KOSPI + 나스닥 + 달러/원 환율 자동 수집 (`FinanceDataReader`)
- 기술적 지표(RSI, MACD, 볼린저밴드, 이동평균 5/20/60일) 자동 계산

### F-02. 뉴스 감성 분석
- 종목별 당일 뉴스 자동 수집 (네이버 모바일 주식 뉴스 API)
- **회사 관련성 우선 정렬** — 종목명이 제목/본문에 포함된 기사를 상위로 노출
- **Claude(Anthropic)**로 뉴스 감성 분석 → 긍정(+1) / 중립(0) / 부정(-1) 점수화
- 감성 점수를 예측 모델의 입력 피처로 활용
- 대시보드에서 각 헤드라인 클릭 시 원문 기사로 이동 (링크 제공)

### F-03. 외국인/기관 순매수 데이터 수집
- KRX(한국거래소) 또는 `pykrx` 라이브러리로 외국인·기관 순매수 수집
- 외국인·기관 연속 순매수/순매도 일수 계산 → 추세 강도 반영

### F-04. 방향 예측 (상승 ↑ / 하락 ↓)
- **RandomForest + XGBoost 앙상블** 모델로 이진 분류
- 예측 결과: 상승/하락 + 신뢰도(확률) %
- 기준: 내일 종가 > 오늘 종가 → 상승, 반대 → 하락

### F-05. 가격대 예측
- **상승 시나리오:** 예상 목표가 범위 (하단 ~ 상단)
  - 예) "상승 시 74,500원 ~ 76,200원 구간 도달 예상"
- **하락 시나리오:** 예상 지지선 범위 (하단 ~ 상단)
  - 예) "하락 시 71,800원 ~ 72,500원 구간에서 지지 예상"
- 계산 방식: 볼린저밴드 + ATR(평균 실제 변동폭) + 피봇 포인트 기반

### F-06. 애널리스트 리포트 자동 생성
- 매일 종목별 **PDF 애널리스트 보고서** 자동 생성
- 구성: 종합 의견 → 예측 결과 → 가격대 분석 → 기술적 지표 → 뉴스 감성 → 외국인/기관 동향
- 전문적 어조 + 투자 의견 (강력매수 / 매수 / 중립 / 매도)
- Claude(Anthropic)가 종합 코멘트 문장 자동 작성 (실패 시 규칙 기반 대체)

### F-07. 이메일 자동 전송
- 매일 오후 **4시 30분** (장 마감 후) 자동 실행
- 이메일로 종목별 예측 요약 본문 + PDF 리포트 첨부 전송
- 전송 형식: HTML 이메일 본문 (종목별 요약 카드) + PDF 첨부파일
- 수신자 목록 설정 가능 (`config.yaml` 에서 관리)
- 전송 실패 시 5분 후 재시도 (최대 2회)

### F-08. 모델 자동 재학습 스케줄러
- **매주 일요일 새벽 2시** 자동 재학습 실행
- 최근 데이터 추가 후 모델 업데이트 (전 종목)
- 재학습 후 정확도 로그 기록
- 학습 결과는 로그로 보고

### F-09. 웹 대시보드
- 종목 선택 → 예측 결과 + 차트 + 리포트 확인
- 최근 30일 예측 적중률 히스토리 시각화
- PDF 리포트 다운로드 버튼

## 7. 제약 사항

- 뉴스 수집: 네이버 모바일 뉴스 API 응답 구조 변경 시 수집 방식 변경 필요 가능
- 외국인/기관 데이터: `pykrx` 라이브러리, KRX 서버 상태에 의존
- Claude API: 감성 분석 + 리포트 문구 생성에 비용 발생 (소액)
- 가격대 예측은 **통계적 범위** 제시이며, 보장된 목표가 아님
