1. 개요

본 문서는 StockSense 웹 대시보드의 화면 구성, 화면별 UI 요소, 사용자 이벤트, 연동 API를 정의한다. 모든 화면은 단일 페이지(SPA 형태)로 동작하며, 종목 선택 후 예측·백테스트·리포트 기능을 탭과 버튼으로 제공한다.

1.1 화면 흐름

메인 대시보드→ 종목 선택 → 예측 결과↹ 탭 전환 ↹ 백테스트 기록→ PDF → 리포트 다운로드

1.2 화면 목록

화면 ID화면명설명연동 API
SCR-001메인 대시보드종목 선택 그리드 및 기능 진입 영역GET /
SCR-002예측 결과내일 주가 방향·신뢰도·가격대·뉴스 감성·코멘트GET /api/predict/{code}
SCR-003백테스트 기록과거 예측 정확도 및 일자별 적중 내역GET /api/history/{code}
SCR-004PDF 리포트애널리스트 리포트 PDF 생성·다운로드GET /api/report/{code}

1.3 공통 UI 규칙

  • 테마: 다크 모드 (배경 #0F172A, 패널 #1E293B, 강조 #3B82F6)
  • 색상 의미: 빨강 = 상승, 파랑 = 하락 (국내 증시 관례 적용)
  • 반응형: 종목 그리드·통계 카드는 화면 폭에 따라 자동 줄바꿈 (CSS Grid auto-fill)
  • 상단 헤더(로고·부제)는 전 화면 공통 고정

5. SCR-004 · PDF 리포트

[📄 PDF] 버튼 클릭 시 GET /api/report/{code}?generate=true를 호출하여 서버에서 애널리스트 리포트 PDF를 생성하고 즉시 다운로드한다.

No.항목내용
1트리거기능 패널의 [📄 PDF] 버튼 → downloadReport() 실행
2요청GET /api/report/{code}?generate=true
3서버 처리report_generator가 예측·지표·감성·차트를 종합해 PDF 생성
4응답PDF 파일 다운로드. 파일명: {code}_report.pdf

6. 연동 API 요약

메서드엔드포인트화면설명

GET/api/predict/{code}SCR-002단일 종목 즉시 예측 (주가·뉴스·감성·코멘트)

GET/api/report/{code}SCR-004PDF 리포트 생성·다운로드
