```
│  오늘 종가:  74,200원                              │

```

**투자 의견 기준:**

| 조건 | 투자 의견 |
|------|---------|
| 신뢰도 75% 이상 + 상승 | 강력매수 (Strong Buy) |
| 신뢰도 60~75% + 상승 | 매수 (Buy) |
| 신뢰도 50~60% 또는 혼재 신호 | 중립 (Neutral/Hold) |
| 신뢰도 60~75% + 하락 | 매도 (Sell) |
| 신뢰도 75% 이상 + 하락 | 강력매도 (Strong Sell) |

### SECTION 6: AI 종합 코멘트

```
■ AI 종합 분석 의견

  삼성전자는 오늘 장에서 74,200원에 마감하며 전일 대비 1.2% 상승세를
  이어갔다. 기술적 관점에서 RSI(58.2)는 과매수 구간에 진입하지 않은
  상태에서 MACD가 양전환을 유지하고 있어 단기 상승 모멘텀이 살아있는
  것으로 판단된다.

  수급 측면에서는 외국인 투자자가 3일 연속 순매수(+1,230억원)를 기록
  하며 추세적 매수 흐름을 보이고 있다. 특히 오늘 기관도 동반 순매수로
  전환하며 스마트머니의 긍정적 포지션 확인이 가능하다.

  뉴스 감성 역시 HBM3E 납품 협상 소식 등 긍정적 재료가 우위를 점하며
  전체 감성 점수 +0.42를 기록했다.

  종합적으로 내일 주가는 74,800원~76,500원 구간으로의 단기 상승 가능성이
  높은 것으로 AI 모델이 판단하나, 중국 반도체 규제 관련 뉴스는 하락
  리스크 요인으로 모니터링이 필요하다.

  → 투자 의견: 매수 (Buy), 단기 목표가 75,200원~76,500원
```

## 5. 리포트 생성 기술

| 항목 | 기술 |
|------|------|
| PDF 생성 | reportlab (Python) |
| HTML → PDF | Edge 헤드리스 (백업) |
| AI 코멘트 작성 | Google Gemini (gemini-2.5-flash) |
| 차트 이미지 | matplotlib → PNG → PDF 삽입 |
| 이메일 전송 | smtplib + email (Python 표준 라이브러리, 추가 설치 불필요) |
